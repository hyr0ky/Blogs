## 变量类型


### 基本类型
- 数值类型：
	- uint  无符号整数；uint8 是8位无符号整数，取值范围0-255
	- int 有符号整数
- 布尔类型： true / false
- ==地址类型==：存储以太坊地址；标识账户，涉及到转账，权限控制等核心功能


### ==复杂类型==

- 数组： 分为固定大小数组和动态数组。固定大小数组如`uint[5]`表示包含 5 个uint类型元素的数组，动态数组如`uint[]`，其长度可以在运行时动态改变
- 结构体：struct允许开发者自定义复杂的数据结构，将不同类型的数据组合在一起。
- 映射： mapping类似于其他编程语言中的哈希表，用于建立键值对的映射关系。在智能合约中，常用来存储账户余额等信息，比如mapping(address => uint256) public balances;表示建立了一个从地址到无符号整数（可用于存储余额）的映射，这是非常常用且重要的复杂类型，要深入理解其原理和使用方法。

## 概念

- 通过自动化生成并执行大量的随机测试用例来发现产品或协议的未知漏洞


### 前置知识

- 内存破坏漏洞原理
- 逆向能力
- 调试能力
## 原理

```bash
data ---> 进程 ---> output


(自动化)Fuzz    ->
		1. 生成Data 
			1. 随机
			2. 基于模板DATA生成Fuzz
			   
		2.基于覆盖制导Fuzz   -> 感知进程 -> 路径反馈 —> 基于反馈 样本变异  
			input   : yes / no
			通过DATA输入，感知进程，探测路径，再基于路径反馈 进行样本变异	   
			if  进程标志1 { 路径1 } else 进程标致2 { 路径2 }


基于HTTP理解Fuzz   --->  基于模板的DATA生成 
	固定不变   : HTTP 1.0
	有限变动   ： GET /POST /DELET。。。   
	任意变动   ：xxxx.html   (DATA)
```

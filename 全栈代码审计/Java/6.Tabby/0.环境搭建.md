## 适用范围
> Codeql分析链子好难受，不能跨库查 ，污点分析流也很乱（我太菜了）不知道这个能不能改善

- 挖掘目标项目中的反序列化利用链，支持大多数序列化机制，包括 Java 原生序列化机制、Hessian、XStream 等
- 挖掘目标项目中的常见 Web 漏洞，支持分析 WAR/JAR/FATJAR/JSP/CLASS 文件
- 搜索符合特定条件的函数、类，譬如检索调用了危险函数的静态函数
